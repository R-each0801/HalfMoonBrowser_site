# 15. データ管理

設定 → **「データ管理」** から、以下のデータを一括で管理できます。

## パスワード

### 保存

- トグル「**パスワードを保存**」を ON にすると、ログイン時に保存するか
  確認ダイアログが出ます
- 保存されたパスワードは **端末内のみ** に暗号化して保管されます
  （iCloud Keychain は未使用）

### 確認・編集

**「保存されたパスワード」** をタップ → Face ID / Touch ID / パスコード認証後、
保存一覧が表示されます。

- サイトごとにユーザー名 / パスワードを確認
- 不要なものは削除

![保存パスワード一覧](/screenshots/s40-passwords.png)

## サイトデータを管理

「**サイトデータを管理**」で、**Cookie / キャッシュ / LocalStorage / IndexedDB**
をサイト単位で削除できます。

- サイトごとのサイズが表示されます
- スワイプで個別削除
- 一括削除ボタンあり

![サイトデータ一覧](/screenshots/s41-site-data.png)

## ダウンロード

### ダウンロード操作

ページ上のファイルリンク（PDF / ZIP / 画像 / 動画など）を長押し →
「ダウンロード」で保存します。ダウンロード中は画面右下に進捗バナーが出ます。

### 一覧

「**ダウンロード**」→ 進行中 + 完了済みのファイル一覧。

- タップで開く / 共有シートへ
- スワイプで削除
- ダウンロード完了後は、iOS の「ファイル」アプリの
  **「HalfMoon」フォルダ** からもアクセスできます

![ダウンロード一覧](/screenshots/s42-downloads.png)

## 履歴

### 表示

「**履歴を表示**」→ 日付別にグループ化された履歴一覧。

- タップで開き直す
- スワイプで個別削除
- 日付ごとの「その日を全削除」

### 一括削除

「**履歴をすべて削除**」で、全履歴を消去。確認ダイアログが出ます。

### プライベート

プライベートブラウザ中の閲覧は **履歴に一切記録されません**。
