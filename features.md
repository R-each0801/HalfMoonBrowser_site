# 機能一覧

HalfMoon は、ブラウザとして必要な機能をしっかり揃えながら、
片手で使える快適さを損なわないよう設計されています。

## 広告ブロック

URL フィルタ、CSS 非表示、広告検知バイパスを組み合わせた **3 層構造** で
広告と追従ウィジェットを自動で除去します。

## プライベートブラウザ

Face ID / Touch ID でロック。閲覧履歴・Cookie・ログイン情報を端末に残さず、
バックグラウンドからの復帰時にも都度再認証します。

Pro 版では **プライベート専用のお気に入り** を通常モードと分離できます。

## データ管理

- **パスワード**: 端末内 Keychain に暗号化保存、iOS 標準の自動入力とも連携
- **サイトデータ**: Cookie / キャッシュ / LocalStorage をサイト単位で削除
- **ダウンロード**: PDF / ZIP / 画像を iOS「ファイル」App に保存
- **履歴**: 日付別に整理。プライベートでの閲覧は記録しない

## iOS との連携

| 機能 | できること |
|---|---|
| iCloud 同期 | 設定を他の iPhone / iPad と自動同期 |
| 標準ブラウザ指定 | iOS 設定から HalfMoon をデフォルトに |
| 共有シート | Safari 等から「HalfMoon で開く」 |
| URL スキーム | `halfmoon://open?url=...` で他アプリから起動 |
