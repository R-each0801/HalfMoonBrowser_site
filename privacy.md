# HalfMoon Browser プライバシーポリシー

最終更新日: 2026 年 4 月 21 日

---

本プライバシーポリシー（以下「本ポリシー」といいます）は、HalfMoon Browser
（以下「本アプリ」といいます）の提供者（以下「当方」といいます）が、ユーザーの
個人情報およびプライバシーをどのように取り扱うかを定めるものです。

## 要点（TL;DR）

- **当方のサーバーには、一切のユーザー情報を送信しません。**
- 閲覧履歴・ブックマーク・パスワード等は **端末内のみ** に保存されます。
- iCloud 同期を ON にした場合、**Apple の iCloud** に設定のみが保存されます
  （当方はアクセスできません）。
- 分析ツール・クラッシュレポート SDK・広告 SDK は **使用していません**。
- 表示した Web サイトが行う通信は、各サイトのプライバシーポリシーに従います。

以下、詳細を記載します。

## 第 1 条（収集する情報）

### 1.1 端末内にのみ保存される情報

以下は **本アプリが端末のローカルストレージ** に保存するものです。当方のサーバーに
送信されることはありません。

| 項目 | 保存先 | 用途 |
|---|---|---|
| 閲覧履歴 | アプリ内 UserDefaults | 履歴表示 / 候補表示 |
| 開いているタブ | アプリ内 JSON ファイル | タブ復元 |
| お気に入り（スタートページ） | アプリ内 JSON ファイル | お気に入り表示 |
| 保存済みパスワード | Keychain（端末暗号化） | 自動入力 |
| 設定（検索エンジン / ダイヤル配置等） | UserDefaults | 設定保持 |
| 非表示にした広告セレクタ | UserDefaults | ユーザー非表示 |
| ダウンロードファイル | アプリ内 Documents フォルダ | ダウンロード一覧 |
| サイトごとのページ倍率 | UserDefaults | ズーム保持 |

### 1.2 サイト運営者側の Cookie / LocalStorage

本アプリは Web ブラウザですので、ユーザーが訪問した Web サイトがブラウザに
保存する Cookie / LocalStorage / IndexedDB 等は、**iOS の WKWebView が管理する
ストレージ** に保存されます。

これらは:

- プライベートブラウザでは **セッション終了時に自動削除** されます
- 通常モードでは保持され、設定 → データ管理 → 「サイトデータを管理」から
  削除できます

### 1.3 Face ID / Touch ID

プライベートブラウザの本人確認に使用します。**生体認証データは iOS が管理** し、
アプリは認証の成否のみを受け取ります（本アプリが生体情報を取得することはありません）。

### 1.4 写真・カメラへのアクセス

- **写真**: 画像長押しで「写真に保存」を選択した時のみ、写真ライブラリへの
  追加を行います
- **カメラ**: URL バーの QR コード読み取りを選択した時のみ、カメラを起動します

取得された画像 / QR データは、アプリ内のみで処理され、外部送信されません。

## 第 2 条（外部送信）

本アプリから **当方のサーバーへのデータ送信は一切ありません**。

ユーザーの操作によって以下の第三者サービスと通信が発生することがあります:

### 2.1 表示した Web サイト

ユーザーが開いた URL には通常通り HTTP / HTTPS リクエストが送られます。これは
ブラウザとしての本質的動作です。

### 2.2 選択した検索エンジン

設定した検索エンジン（Google / DuckDuckGo / Bing 等）に検索クエリが送信されます。
送信内容・取扱いは各検索エンジンのプライバシーポリシーに従います。

### 2.3 iCloud（同期を ON にした場合）

設定 → iOS → iCloud 同期を ON にすると、iOS の **CloudKit API** 経由で
**設定情報のみ** が Apple の iCloud に保存されます。当方のサーバーは経由しません。

同期対象:
- 既定の検索エンジン
- ダークモード / スーパーダーク等の表示設定
- 広告ブロックの ON / OFF 設定
- 触覚フィードバックの設定

**閲覧履歴・お気に入り・パスワード・タブ情報は同期されません。**

### 2.4 App Store / StoreKit（課金時）

Pro 版購入時は、Apple の StoreKit API を通じて App Store に購入リクエストが
送られます。課金情報は Apple が管理し、当方は購入が完了したか否かの情報のみを
受け取ります。

### 2.5 Google 翻訳（ダイヤル「翻訳」を実行した場合）

ダイヤルの「翻訳」アクションを選択すると、現在の URL を `translate.google.com`
に渡してブラウザで翻訳版を開きます。翻訳処理は Google 側で行われ、当方は
関与しません。

## 第 3 条（広告ブロック機能について）

本アプリの広告ブロックは **完全にオフライン** で動作します:

- 広告配信ドメインのリストはアプリにバンドルされており、外部から動的に
  ダウンロードしません
- 広告を非表示にする CSS セレクタも内蔵されています
- ユーザーが手動で非表示にしたセレクタは **端末内のみに保存** されます

広告ブロックに関するログやテレメトリは、当方のサーバーに送信しません。

## 第 4 条（分析ツール・SDK の不使用）

本アプリは以下を **使用していません**:

- Google Analytics / Firebase Analytics 等の分析ツール
- Crashlytics 等のクラッシュレポート SDK
- 広告 SDK / 広告 ID（IDFA）取得
- ソーシャルログイン SDK

## 第 5 条（不具合レポートについて）

不具合発生時のレポートは、**ユーザーが明示的にコピー・送信** しない限り、
当方および第三者のサーバーには送信されません。

レポートには閲覧中の URL、端末モデル、iOS バージョン等が含まれます。
お送りいただく際は、個人を特定できる情報が含まれていないか事前にご確認ください。

## 第 6 条（子供のプライバシー）

本アプリは、13 歳未満の児童から故意に情報を収集することはありません。なお、
本アプリは Web ブラウザとして一般向けに提供されており、特定年齢層をターゲット
としていません。

## 第 7 条（セキュリティ）

- 保存済みパスワードは iOS の Keychain 暗号化ストレージに保存されます
- 通信は HTTPS を推奨しますが、ユーザーが指定した HTTP URL も開けます
- 本アプリの脆弱性を発見された場合、以下の連絡先までご報告ください
  （責任ある開示にご協力ください）

## 第 8 条（本ポリシーの変更）

本ポリシーは、法令の変更、機能追加、第三者サービスとの連携変更等に伴い、
予告なく改定されることがあります。重要な変更がある場合は、本アプリの
アップデート時に告知します。
