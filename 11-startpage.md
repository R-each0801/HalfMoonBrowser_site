# 11. スタートページを整える

## スタートページとは

新しいタブや、ホーム（右ダイヤル）を選んだときに表示される **「お気に入り一覧」**
の画面です。

構成:

1. 上部の **検索バー**（タップで URL / 検索入力）
2. **タブ一覧**（横スクロールのストリップ）
3. **お気に入りタイル**（6 タイル / 行のグリッド）
4. 右下の **フローティング歯車**（設定を開く）

![スタートページ全景](/screenshots/s32-startpage-full.png)

## タイルを追加する

### 方法 1: ダイヤルから

開いているページで **左ダイヤル → ブックマーク追加** → 自動でスタートページに
タイルが追加されます（→ [06. ブックマーク追加](/06-practice-bookmark-scroll)）。

### 方法 2: スタートページで手動追加

右上の「**編集**」ボタン → 「＋」タイル → URL とタイトルを入力。

![編集モードと ＋ タイル](/screenshots/s33-startpage-edit.png)

### 方法 3: 他ブラウザから読み込む

設定 → スタートページ → **「ブックマークを読み込む」** で、他ブラウザから
エクスポートした HTML を読み込めます。

対応形式:
- Chrome / Safari / Firefox の「ブックマーク書き出し」HTML

## フォルダを作る

**編集** モードでタイルを **長押し** → 「フォルダへ移動」→ 新規フォルダ名入力。
タイルを他のタイルに重ねる（ドラッグ）と自動でフォルダができます。

![フォルダ作成後](/screenshots/s34-startpage-folder.png)

## タイルの操作

| 操作 | 結果 |
|---|---|
| タップ | そのサイトを開く |
| 長押し | コンテキストメニュー（新しいタブで開く / フォルダへ移動 / 削除） |
| 編集モード中の長押し＋ドラッグ | 並び替え |
| 編集モード中の赤丸 | 削除 |

## アイコンについて

サイトアイコンは **apple-touch-icon → Google Favicon API** の順で取得します。
取得できない場合は 🌐 地球マークが表示されます。

## 検索バー / タブ一覧の表示切替

- 検索バーを隠したい: 設定 → スタートページ → **「検索バーを表示」** OFF
- タブ一覧を隠したい: 設定 → スタートページ → **「タブ一覧を表示」** OFF

## 検索履歴の候補表示

検索バーをタップしたとき、直近の検索 / URL を候補として出すかの切替:

- 設定 → スタートページ → **「検索履歴を候補に表示」** ON / OFF

プライベートブラウザでの入力は **記録されません**。

## 既定の検索エンジン

検索バーに文字を入れて確定したときに使う検索サイトを選択:

- 設定 → スタートページ → **「既定の検索エンジン」** Picker
- Google / DuckDuckGo / Bing など
