# 09. 実践: ページ内検索をする

## 🎯 この実践で覚えること

サブメニューの **🔍 ページ内検索** で、開いているページの中から文字を探す機能。

## 手順

### ステップ 1: 記事や長文のページを開く

### ステップ 2: サブメニュー → 「ページ内検索」

サブメニュー → **🔍 ページ内検索** を選ぶと、画面上部に入力欄が表示されます。

![ページ内検索バー](/screenshots/s27-findinpage-bar.png)

### ステップ 3: 検索ワードを入力

入力すると該当箇所が **ハイライト** されます。

- **↑ ↓ ボタン**: 前後のマッチへ移動
- **件数表示**: 現在 n 件中 m 番目
- **× ボタン**: 閉じる

![ハイライトされた検索結果](/screenshots/s28-findinpage-matches.png)

### ステップ 4: 閉じる

× ボタンをタップ、またはキーボードの return キーで確定後にもう一度検索バーの
× で閉じます。

## 💡 Tips

- **大文字小文字の区別なし**（標準の WKWebView 検索仕様）
- **正規表現は使えません**（iOS の API 制限）
- 画像や iframe の中の文字は検索対象外です

## ✅ できるようになったか確認

- [ ] サブメニューから検索バーを出せる
- [ ] ハイライトで目的の箇所にジャンプできる
- [ ] 閉じられる
