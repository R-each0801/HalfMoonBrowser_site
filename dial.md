# ダイヤル操作

画面の **左右の端** からスライドするだけで、半円メニューが展開します。
下部ツールバーのように画面を圧迫せず、親指の届く場所で操作が完結します。

## 初期配置

| 左ダイヤル (上から) | 右ダイヤル (上から) |
|---|---|
| ⬆ ページ最上部 | 🔍 検索・URL |
| ➕ 新規タブ | ← 戻る |
| ⋯ サブメニュー | 🏠 ホーム |
| 🔖 ブックマーク追加 | → 進む |
| ⬇ ページ最下部 | × タブを閉じる |

## サブメニュー

左ダイヤル中央の **「サブメニュー」** を選ぶと、縦リストが展開されます。
よく使わないが便利な項目はここに集約:

- 🗂 タブ一覧
- ⬆ 共有
- 🔍 ページ内検索
- 👆 広告を消す (ピッカー起動)
- 📜 自動スクロール
- ⚙ 設定

## カスタマイズ

設定 → **「ダイヤル編集」** から、各項目を追加・削除・並び替えできます。
最大 **6 項目**（Pro 版は **12 項目**）まで配置可能。

利用できる主なアクション:

- 戻る / 進む / 更新
- 検索・URL / 新規タブ / タブを閉じる / タブ移動 / タブ一覧
- ホーム / ブックマーク追加 / 履歴
- 共有 / ページ内検索 / 翻訳 / リーダー
- ページ最上部 / 最下部 / 自動スクロール
- 拡大 / 縮小 / PC 表示切替
- 広告を消す / サイト別広告ブロック
- スーパーダーク / プライベート切替
