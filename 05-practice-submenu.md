# 05. 実践: サブメニューを開いてみる

## 🎯 この実践で覚えること

左ダイヤルにある「サブメニュー」の使い方。ダイヤル本体に乗り切らない
**あまり使わないが時々必要な** 操作をここにまとめます。

## サブメニューとは

左ダイヤルの **上から 3 番目** にある「サブメニュー」を選ぶと、**縦長のリスト**
が画面中央に展開されます。初期状態では以下が並んでいます:

- 🗂 **タブ一覧**
- ⬆ **共有**
- 🔍 **ページ内検索**
- 👆 **広告を消す**
- 📜 **自動スクロール**
- ⚙ **設定**

![サブメニュー展開中](/screenshots/s14-submenu-open.png)

## 手順（ホールド＆リリース の場合）

### ステップ 1: 左端にタッチ

左ダイヤルが展開します（上から 5 項目が半円状に並ぶ）。

### ステップ 2: 「サブメニュー」まで上下スライド

上から 3 番目の **サブメニュー** に指を動かしてハイライト。

![「サブメニュー」ハイライト中](/screenshots/s15-submenu-highlight.png)

### ステップ 3: 指を離さないまま縦リストへ

**指を離すとサブメニューが開き、そのままリスト内の項目までスライドして選択できます**。

![リリース直後 → 縦リストを上下選択中](/screenshots/s16-submenu-list-slide.png)
試しに **🗂 タブ一覧** まで動かして離してみてください。
現在開いているタブの一覧シートが出ます。

![タブ一覧シート](/screenshots/s17-tab-list-sheet.png)

## 項目説明

| 項目 | 内容 |
|---|---|
| 🗂 タブ一覧 | 開いているタブのサムネ一覧。タップで切替、長押しで閉じる。 |
| ⬆ 共有 | 標準の iOS 共有シートを開く。他アプリに URL 送信。 |
| 🔍 ページ内検索 | ページ内の文字を検索（Command-F 相当）。 |
| 👆 広告を消す | 次のタップでその要素を非表示にするピッカーを起動。 |
| 📜 自動スクロール | ページを一定速度で自動スクロール開始。 |
| ⚙ 設定 | 設定画面を開く。 |

## 💡 サブメニューの並びはカスタマイズできる

よく使う項目を増やしたい / 不要な項目を外したいときは、設定 → ダイヤル編集 →
「サブメニュー」セクションから項目を追加・削除・並び替えできます
（→ [10. ダイヤルを編集する](/10-dial-customize)）。

## タップ切替モードの場合

- 左端タップ → ダイヤル表示（閉じずに固定）
- 「サブメニュー」タップ → 縦リストが展開
- リスト項目タップ → 実行

## ✅ できるようになったか確認

- [ ] サブメニューを開ける
- [ ] 縦リストから「タブ一覧」を実行できる
- [ ] サブメニュー内の各項目の役割がわかった
