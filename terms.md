# HalfMoon Browser 利用規約

最終更新日: 2026 年 4 月 21 日

---

本利用規約（以下「本規約」といいます）は、HalfMoon Browser（以下「本アプリ」と
いいます）の提供条件および本アプリの利用者（以下「ユーザー」といいます）と
本アプリ提供者（以下「当方」といいます）との間の権利義務関係を定めるものです。
本アプリをインストール・起動した時点で、本規約に同意したものとみなされます。

## 第 1 条（適用範囲）

1. 本規約は、ユーザーと当方との間の本アプリの利用に関わる一切の関係に適用されます。
2. 当方は、本規約のほか、利用方法を定めたマニュアル等を、本アプリに付随する
   別途の文書として提供することがあります。これらは本規約の一部を構成するものと
   します。

## 第 2 条（利用条件）

1. 本アプリは、Apple Inc. の App Store を通じて配布されます。App Store の
   利用規約（Apple Media Services 利用規約等）に同意できない場合、本アプリは
   利用できません。
2. 本アプリは、iOS / iPadOS 16.0 以上の端末で動作します。それ以外の環境での
   動作は保証しません。

## 第 3 条（ライセンス）

1. 当方は、本規約に従うことを条件として、ユーザーに対して、本アプリを個人的に、
   非商用で利用する、非独占的、非譲渡的、再許諾不可のライセンスを許諾します。
2. ユーザーは、本アプリの逆コンパイル、リバースエンジニアリング、改変、
   配布、再販売を行ってはなりません。

## 第 4 条（Pro 版機能）

1. 本アプリには、App Store 経由のアプリ内課金（買い切り）により解放される
   Pro 版機能があります。
2. Pro 版機能の課金は App Store の規定に従います。キャンセル・返金については
   Apple Inc. の定める手続きに従ってください。当方は直接の返金には応じられません。
3. ユーザーは、同一の Apple ID でサインインしている複数端末間で Pro 版機能を
   共有できます。

## 第 5 条（禁止事項）

ユーザーは、本アプリの利用に際し、以下の行為を行ってはなりません:

1. 法令、公序良俗に反する行為
2. 第三者の権利（著作権、商標権、プライバシー等）を侵害する行為
3. 本アプリを利用して違法なコンテンツにアクセスする行為
4. 本アプリのシステムに過度な負荷をかける、または不正な手段で機能を利用する行為
5. 本アプリを改変・解析・複製して第三者に配布する行為
6. その他、当方が不適切と判断する行為

1. 本アプリは Web ブラウザであり、ユーザーが指定した URL の Web コンテンツを

   それぞれのコンテンツ提供者が責任を負い、当方は一切の責任を負いません。
2. 本アプリが表示する検索結果・候補は、ユーザーが選択した検索エンジンから
   取得したものであり、当方が選定・編集するものではありません。

## 第 7 条（広告ブロック機能について）

1. 本アプリは、ユーザーの閲覧体験向上を目的として、既知の広告配信ドメインの

2. 広告ブロックは、Web サイト運営者の収益モデルに影響を与える可能性があります。

3. 広告ブロックの有効 / 無効はユーザー自身の判断で設定するものであり、その

4. 本アプリがブロックしきれない広告、または誤って非広告要素をブロックする場合が
