# 10. ダイヤルを編集する

## カスタマイズの入口

設定 → **「ダイヤル」→ ダイヤルを編集** から開きます。

![ダイヤル編集画面](/screenshots/s29-dial-edit.png)

## 画面構成

3 つのセクションがあります:

- 「**左ダイヤル**（N 項目）」
- 「**右ダイヤル**（N 項目）」
- 「**サブメニュー**（N 項目）」

各セクションにはすでに配置されている項目が並び、末尾に **「＋ 項目を追加」**
ボタンがあります。

## 項目の並び替え

各項目の **右端にあるハンバーガーアイコン** （横三本線）が、並び替え可能の
目印です。

![項目の右端ハンバーガー](/screenshots/s30-drag-handle.png)
項目を **長押ししてドラッグ** すると上下に並び替えできます（SwiftUI の List 標準動作）。

## 項目の削除

- 項目を **左へスワイプ** → 「削除」ボタンが出る
- または上部の「編集」ボタン → 左端の 🔴 をタップ → 「削除」

## 項目の追加

「**＋ 項目を追加**」をタップすると、選択可能なダイヤルアクション一覧が出ます。

![項目追加ピッカー](/screenshots/s31-action-picker.png)

### すでに追加済みの項目は

グレーアウトされた状態で表示され、**右側に赤色の ⊖ マーク** が付いています。
**タップすると逆に削除** されます（誤タップ時の取り消しが直感的）。

## 最大項目数

- 通常版: **各 6 項目まで**
- Pro 版: **各 12 項目まで** (→ [17. Pro 版](/17-pro))

## 便利なボタン

画面下部にある:

- **「左右のダイヤルを入れ替える」**: 右利き / 左利きを切替えたいとき
- **「デフォルトに戻す」**: 初期配置に戻す

## ✅ おすすめカスタム例

### 片手操作重視

| 位置 | おすすめ項目 |
|---|---|
| 左上から | ページ最上部 / ページ最下部 / サブメニュー / 新規タブ / 戻る |
| 右上から | 検索・URL / 進む / ホーム / タブ一覧 / タブを閉じる |

### タブ多用派

| 位置 | おすすめ項目 |
|---|---|
| 左上から | 新規タブ / タブ一覧 / タブ移動 / ブックマーク追加 / サブメニュー |
| 右上から | 検索・URL / 戻る / ホーム / 進む / タブを閉じる |

### 広告ブロック重視

サブメニューに「広告を消す」「サイト別広告ブロック」の両方を入れておくと、
読みづらいサイトに出会ったときに切替しやすくなります。
