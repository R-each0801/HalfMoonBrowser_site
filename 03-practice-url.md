# 03. 実践: URL・検索バーを呼び出す

## 🎯 この実践で覚えること

**右ダイヤルのいちばん上「検索・URL」** を使って検索バーを呼び出す流れ。

これは HalfMoon でいちばんよく使う操作です。

## 手順

### ステップ 1: 右端をタッチ

画面の **右端**（右辺）に指を置きます。

![右端にタッチした瞬間](/screenshots/s07-touch-right-edge.png)
右ダイヤルが半円状に展開されます。上から順に:

1. 🔍 **検索・URL** ← 今回これを使います
2. ← 戻る
3. 🏠 ホーム
4. → 進む
5. × タブを閉じる

### ステップ 2: いちばん上までスライド

指を離さず、そのまま **上** に動かします。一番上の「検索・URL」が
ハイライト表示され、軽い振動が返ってきます。

![「検索・URL」がハイライト中](/screenshots/s08-urlinput-highlight.png)

### ステップ 3: 指を離す

ハイライトされた状態で指を離すと、**検索バー** が画面上部に現れて
キーボードが開きます。

![検索バー表示 + キーボード](/screenshots/s09-search-bar-open.png)

### ステップ 4: 入力して確定

- **URL を直接入力**: `github.com` など → 移動
- **キーワードで検索**: `アンテナ速報` など → 既定の検索エンジンで検索
- **過去の履歴 / お気に入り** が候補表示されるのでタップでも OK

入力後 **Enter** または候補タップで読み込み開始。

## 💡 ワンポイント

- **タップ切替モード** の場合: 右端タップ → ダイヤルから「検索・URL」をタップ。
- 検索エンジンは設定 → 「スタートページ」→「既定の検索エンジン」で変更できます
  （Google / DuckDuckGo / Bing など）。
- 検索履歴を候補に出したくない場合は、同じ画面の「検索履歴を候補に表示」を OFF に。
- **プライベートブラウザ中の入力は履歴に残りません。**

- [ ] URL / 検索ワードどちらでもページを開ける
