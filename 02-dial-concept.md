# 02. ダイヤルの基本（必読）

## ダイヤルとは

HalfMoon は、画面下部に固定ツールバーを置かず、**画面の左右の端から指で引き出す
半円状のメニュー** ＝ **ダイヤル** で操作するブラウザです。

![右ダイヤル展開中の静止画](/screenshots/s04-right-dial-open.png)

## 操作モードの違い

ダイヤルには **2 つの操作モード** があります。どちらも同じメニューが出ますが、
**項目を確定するやり方** が異なります。設定で好きなほうを選べます。

### ① ホールド＆リリース（初期値）

1. 画面の左端または右端に **指を置いたまま** にする → ダイヤルが展開
2. そのまま指を **上下に動かす** → 項目がハイライトされる（軽い振動）
3. 使いたい項目で **指を離す** → その項目が実行される
4. 指をダイヤルの外に離す、または下端/上端まで動かすと **キャンセル**

> 💡 **Tips**: 指を離す前に画面の中央より内側へ動かすと、キャンセル扱いになります。
> 誤タップ予防になります。

### ② タップ切替

1. 画面の左端または右端を **タップ** → ダイヤルが出て開きっぱなしになる
2. 項目を **タップ** → 実行
3. ダイヤル外を **タップ** → 閉じる

> 💡 **Tips**: 両手でゆっくり使いたい / 長押しが苦手な方におすすめ。

### モードの切替え方

右下の歯車 → 「ダイヤル」→「操作モード」 で切替えできます。

![設定「操作モード」の切替 Picker](/screenshots/s05-interaction-mode.png)

## サイズ調整

ダイヤルは **0.8x 〜 1.8x** でサイズを調整できます。画面の大きさに合わせて
お好きな大きさで使ってください。

右下の歯車 → 「ダイヤル」→「サイズ」スライダー。

![設定「サイズ」スライダー](/screenshots/s06-dial-size.png)

## 触覚フィードバック

項目をハイライト / 確定したときに短く振動します。強さは **弱 / 中 / 強** から
選べます（設定 → ダイヤル → 触覚フィードバック）。
