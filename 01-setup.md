# 01. インストールと初回起動

## 動作環境

| 項目 | 要件 |
|---|---|
| OS | iOS / iPadOS 16.0 以上 |
| 端末 | iPhone / iPad (ユニバーサル対応) |
| 容量 | 約 10 MB |

## インストール

App Store で「**HalfMoon**」を検索し、インストールしてください。

![App Store の HalfMoon ページ](/screenshots/s01-appstore.png)

## 初回起動

起動すると最初に **スタートページ** が表示されます。スタートページは

- 上部の検索バー
- タブ一覧
- お気に入りタイル
- 右下のフローティング歯車（設定）

で構成されています。

![初回起動時のスタートページ](/screenshots/s02-startpage-initial.png)

## ダイヤルの初期配置

初期状態では、画面の **左右の端** を指でなぞる（タッチ）とダイヤルが展開します。

- **左ダイヤル**（上から）: ページ最上部 / 新規タブ / サブメニュー / ブックマーク追加 / ページ最下部
- **右ダイヤル**（上から）: 検索・URL / 戻る / ホーム / 進む / タブを閉じる

![左右ダイヤル展開時の比較図](/screenshots/s03-default-dial.png)

この初期配置は [10. ダイヤルを編集する](/10-dial-customize) でいつでも変更できます。

## 権限の許可

使い始めに以下の権限を求められることがあります。それぞれ **必要なときだけ** 使用されます。

| 権限 | 用途 |
|---|---|
| Face ID / Touch ID | プライベートブラウザの本人確認 |
| 写真ライブラリへの追加 | 画像長押しメニューの「写真に保存」時のみ |
| カメラ | URL バーの QR コード読み取り時のみ |

次のページで、**ダイヤル操作のいちばん大事な基本** を押さえます。
