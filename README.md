# HalfMoon Browser ユーザーマニュアル

> iPhone 向けの片手操作ブラウザ「HalfMoon」の使い方ガイドです。

このマニュアルは **ダイヤル操作を実際に手を動かして覚えてもらう** ことを目的に、
1 ページ 1 アクションで構成しています。順に読みながら、お手元の iPhone で
同じ操作を試してください。

> 📦 **このマニュアルはアプリ本体には含まれません。**
> 別リポジトリへの配置・GitHub Pages での公開を想定しています。
