# 06. 実践: ブックマーク追加とページ内ジャンプ

## 🎯 この実践で覚えること

左ダイヤルにまとめられた **ブックマーク追加 / ページ最上部 / ページ最下部** の
3 つの日常系アクション。

## 実践 1: ブックマーク追加（お気に入りに加える）

### 手順

1. 何かの記事ページを開く（例: Wikipedia のいずれかの項目）
2. **左端** にタッチして左ダイヤルを展開
3. 上から **4 番目**「ブックマーク追加」でハイライト → リリース
4. 画面下部にトーストで「追加しました」と表示される

![ブックマーク追加トースト](/screenshots/s18-add-bookmark-toast.png)

### どこに追加されたか

**スタートページ** の一覧に追加されます。ホームボタン（右ダイヤル中央）で戻ると、
タイルとして並んでいるのが確認できます。

![スタートページに追加されたタイル](/screenshots/s19-new-bookmark-tile.png)

### 💡 追加先フォルダ / 削除

追加したタイルは **長押し** でコンテキストメニューが出て、「フォルダへ移動」
または「削除」ができます。スタートページ右上の「編集」ボタンで一括整理も可能です。

## 実践 2: ページ最上部 / ページ最下部

長い記事を読んでいるとき、**「上」と「下」の端に一発で飛ぶ** のがとても便利です。

### 手順

1. 長いページをスクロールで中途まで読み進める
2. **左端** にタッチして左ダイヤルを展開
3. 一番上 ⬆ **ページ最上部** を選んで離す → ページ先頭に戻る
4. 逆に一番下 ⬇ **ページ最下部** を選ぶ → ページ末尾に飛ぶ

![「ページ最上部」ハイライト](/screenshots/s20-scroll-to-top.png)

### 💡 よく使う例

- SNS / アンテナサイトで「今の位置から最新へ」戻る
- 記事末尾のコメント欄に一気に飛ぶ
- 画像一覧を下まで読み切ったあと、トップに戻る

## ✅ できるようになったか確認

- [ ] 左ダイヤルからブックマークを追加できる
- [ ] スタートページにタイルが並んだのを確認できる
- [ ] ページ最上部 / 最下部で瞬間ジャンプできる
