# 18. よくある質問 / トラブルシューティング

## Q. サイトが白紙（真っ白）になって表示されない

**A.** サイト側の広告ブロック検知機構に引っかかっている可能性があります。

### 対処手順

1. **ダイヤル → サイト別広告ブロック** でこのサイトだけ OFF にして再読み込み
2. それでも白紙なら、設定 → 広告ブロック → **「強力ブロック」** を OFF にする
3. それでもダメなら [問い合わせ](#q-問い合わせ) ください

不具合報告時は **URL と端末モデル / iOS バージョン** を添えていただくと修正が早くなります。

## Q. サイトのレイアウトが崩れる / ボタンが反応しない

**A.** 広告ブロックの CSS 非表示や強力ブロックが、サイトの UI と干渉している可能性。

「強力ブロック」を OFF、またはそのサイトだけ広告ブロックを OFF でお試しください。

## Q. YouTube / Netflix / 画像で色が変になる

**A.** 強制ダークモード ON が原因です。設定 → 表示 → 強制ダークモード OFF に
してみてください。

## Q. 動画の全画面表示で横向きにならない

**A.** 設定 → ページ操作 → **「横動画を全画面で横向きに回転」** を ON に。
iOS の端末ロック（縦固定）は影響しない設定です。

## Q. 広告が消えないサイトがある

**A.** サブメニュー → **「広告を消す」** で手動非表示できます
（→ [07. 広告を消す](/07-practice-adblock-select)）。

広範に使える広告ネットワークの追加については、[問い合わせ](#q-問い合わせ) からお知らせください。

## Q. パスワードを iCloud Keychain に保存したい

**A.** HalfMoon のパスワード保存は端末内のみです。
**iOS 標準のパスワード自動入力** は使えるので、Keychain に保存済みのパスワードは
サイトのログイン欄で自動表示されます。

## Q. ダウンロードしたファイルはどこ？

**A.** iOS の **「ファイル」アプリ → このデバイス内 → HalfMoon** フォルダ。
または 設定 → データ管理 → ダウンロード からもアクセス可能。

## Q. 他のブラウザのお気に入りを取り込みたい

**A.** 設定 → スタートページ → **「ブックマークを読み込む」** から、
Chrome / Safari / Firefox 等で書き出した HTML ファイルを読み込めます。

## Q. ダイヤルが反応しない / 開かない

**A.** 以下を確認:

1. 画面のいちばん端（左辺 / 右辺の 5px 程度）をタッチしているか
2. 設定 → ダイヤル → **操作モード** が「ホールド＆リリース」か「タップ切替」か確認
3. ダイヤルが空になっていないか（項目 0 の状態）。設定 → ダイヤル編集 →
   「デフォルトに戻す」で復元できます
