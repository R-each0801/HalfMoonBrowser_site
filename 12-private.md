# 12. プライベートブラウザ

## プライベートブラウザとは

以下を **この端末に残さず** 閲覧するモードです:

- 閲覧履歴
- Cookie / LocalStorage / キャッシュ（セッション終了時にクリア）
- ログイン情報
- 検索履歴候補

## 切替え方

### 方法 1: ダイヤルに追加

ダイヤル編集から「**プライベート切替**」を好きな位置に追加し、
ダイヤルから切替え。

### 方法 2: タブ一覧から

サブメニュー → タブ一覧 → 画面上部の鍵アイコンで切替え。

![プライベートモード中の画面](/screenshots/s35-private-mode.png)
プライベートモード中は、画面のタブストリップやツールバーの色が
**ダークモード風の濃い色** に変わるので、ひと目で判別できます。

## Face ID / Touch ID による保護

初回にプライベートへ切替えるとき、**Face ID / Touch ID / パスコード** の
認証を求められます。以降、以下のタイミングで再認証が走ります:

- アプリをバックグラウンドから前面に戻したとき
- アプリを起動し直して **「次回起動時にプライベートを再開」** が ON のとき

## お気に入りの分離

Pro 版なら、プライベートモード専用のお気に入り（スタートページ）を
通常と別に保管できます。

- 設定 → プライベートブラウザ → **「プライベート専用のお気に入り」** ON
- 初期値は **OFF（通常と共通）**

## 設定項目

### 「次回起動時にプライベートを再開」

ON にすると、前回アプリを閉じたときにプライベートだった場合、次回起動時も
認証後にプライベートのまま再開します。
